How to use phpMyAdmin

Creating a Simple Database

Want to know how to make a database on phpMyAdmin? Just follow these simple steps!

Prerequisites

To create a database on this tutorial, you will need to have phpMyAdmin installed and securely running. If you do not have phpMyAdmin installed, follow the instructions here.

Step One: Logging into phpMyAdmin and creating a Database

Enter the following into your browser of choice to access your phpMyAdmin:

https://your_domain_or_IP_address/phpmyadmin

The following prompt should pop on in your browser window:

Login to your account using the username and password you created when you installed phpMyAdmin. Once you are logged in, click the “Databases” tab found in the header. In the Database Name input, located below the header, type in the name of your database that you want to create. (Remember to use all lowercase when typing the name!) Next to the name entry is a collation tab, set that to “utf8mb4_general_ci” then click the “Create” button.

Congrats! You have created a Database! Now let’s put some data in there!

Step 2: Creating a Table for your Database

If you haven’t already, click on the “Databases” tab in your header. Then click on the Database you want to create a table for. OR you could just click the name of your Database in the directory found on the left side of your browser page. Once you have clicked on the Database you want to work with, look at the “Create Table” section. Enter the desired name of your table, and enter in the number of columns you would like. For the case of this tutorial, set the name of the table to be “dogs” and set the number of columns equal to 5. Once you have done this, click “Create Table.”

After you click “Create Table,” 5 rows will appear, with a number of different boxes in each row. Each of these rows represents a column in your database. The purpose of what you are seeing on your screen is to set the properties of each column in your data table, by adding to the rows on your screen.

For this tutorial, our 5 columns will be represented by the following: id, breed, size, age, and color. Let’s start with “id.”

In the first row, for “Name” enter “id”, leave the “Type” as “INT” and set the length to 15. Check the box under “A_I.” Dialogue should appear, in there, set both of the fields to “PRIMARY.” This is the index for your table. Each item can be represented by a unique number (of up to 15 characters). The items will automatically increment themselves based on their unique id number. Now, onto the other rows.

Add a name for each of the remaining rows. For this tutorial, add the names “breed”, “size”, “age”, and “color.” Since all of the categories (except “age”) will be represented by words, change the “Type” to “VARCHAR” and set the length to “100.” For the row “age”, set the type to “INT” of 3 digits. Do not change anything else in the rows. After all of the information has been added, click the “Save” button at the very bottom right of your screen.

You should see all of the columns in the table! Now onto adding some data to your table!

Step 3: Adding Data

Click the “SQL” tab found in the header of your browser. Under the text area, click “INSERT.” You should see some SQL code appear in the text input box. It should look like this (if you are following these instructions):

INSERT INTO `dogs`(`id`, `breed`, `size`, `age`, `color`) VALUES ([value-1],[value-2],[value-3],[value-4],[value-5])

To insert data, you need to edit the VALUES section of this code:

VALUES ([value-1],[value-2],[value-3],[value-4],[value-5])

Add values for each bracket of code. For example:

VALUES ("1","Beagle","Small","3","Brown")

After you have added data, click the “Go” button at the bottom of the page. You should get a message that the data has been inserted. Add some more data using the code above, and when you are done, click on the database and look at the data you added.

Step 4: Creating and Adding a Database User

Now we need to add a user that is allowed to edit the database we made. We don’t want anyone else to use our root account. To do this, click the home button, then click on the “User Accounts” tab in the header.

This tab lists all of the current users of databases. Below this list, click the “Add User Account” button. Enter a username and password for this user (remember to record it and make sure the password is strong). After that is done, click the “Go” button.

On the Add Privileges screen, click the “Databases” button and select the database you just made. Click “Go.” Next, on the “Database Specific Privileges” screen, check “Select All” and click “Go.” If you got a message saying it worked, yay! You just created a user for your database!

Conclusion

Congratulations! You have successfully created a database on phpMyAdmin!! PhpMyAdmin is a very useful tool for creating different databases. The databases that you create on this application can be displayed on browsers/websites. Once you have created the databases, you can use code on your server to interact with your interface. You can also use code to print your data out onto your browser!
